# Infinity Scroll with Unsplash API

This project is a simple web application that allows users to explore and scroll through a collection of photos using the Unsplash API. It features infinite scrolling, a search bar to find photos based on keywords, and dynamic loading of new images as the user scrolls down the page.

## Features

- Infinite scrolling: Automatically loads more photos as you scroll down the page.
- Search functionality: Enter keywords in the search bar to find photos related to your query.
- Dynamic loading: New photos are fetched from the Unsplash API in response to user actions.
- Responsive design: Works on both desktop and mobile devices.

## Technologies Used

- HTML
- CSS
- JavaScript
- Unsplash API

## How to Use

1. Clone the repository to your local machine.

2. Open the `index.html` file in your web browser.

3. Explore the collection of photos by scrolling down the page.

4. Use the search bar at the top to enter keywords and find specific photos.

5. Click on a photo to view it on Unsplash.
